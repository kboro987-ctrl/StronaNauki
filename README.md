# Inżynieria finansowa — kompendium i testy

Statyczna strona do nauki inżynierii finansowej (kompendium wiedzy, ściąga wzorów,
testy z punktacją), zbudowana na podstawie wykładów 1–12.

## Uruchomienie lokalne
Otwórz `index.html` w przeglądarce.

## Hosting (GitHub Pages)
Strona jest w pełni statyczna (HTML/CSS/JS). Po wypchnięciu do repozytorium wystarczy
włączyć GitHub Pages: **Settings → Pages → Branch: `main` / folder `/ (root)`**.

## Pliki
- `index.html` — struktura strony i nawigacja
- `styles.css` — wygląd (motyw jasny/ciemny)
- `data.js` — treść kompendium, wzory i baza pytań testowych
- `app.js` — logika (nawigacja, silnik quizu, zapis wyników)
